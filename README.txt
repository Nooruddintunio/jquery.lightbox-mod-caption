[fre-FR]-----------------------------------------------------------------
Version modifiée du plugin jquery.lightbox-0.5.js de Leandro Vieira Pinho 
http://leandrovieira.com/projects/jquery/lightbox/

Permet une légende à la lightbox sans la déclarer dans le texte alternatif de l'image source. 
Exemple : ajouter un lien hypertexte à la lightbox permettant de télécharger l'image.
Voir lightbox_caption_sample.html

[eng-GB]-----------------------------------------------------------------
Modified version of jquery.lightbox-0.5.js by Leandro Vieira Pinho 
http://leandrovieira.com/projects/jquery/lightbox/

Allows you to add a complex caption to the lightbox without declaring it as the alternative text 
of the source image.
Example: add a hyperlink to download the lightbox image.
See lightbox_caption_sample.html
